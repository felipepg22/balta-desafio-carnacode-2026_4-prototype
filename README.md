## 🥁 CarnaCode 2026 - Challenge 04 - Prototype

Hi, I'm Felipe Parizzi Galli and this is the space where I share my learning journey during the **CarnaCode 2026** challenge, organized by [balta.io](https://balta.io). 👻

Here you will find projects, exercises and code I'm developing throughout the challenge. The goal is to get hands-on, test ideas and track my progress in the world of technology.

### About this challenge
In the **Prototype** challenge I had to solve a real problem by implementing the **Design Pattern** in question.
During this process I learned:
* ✅ Software Best Practices
* ✅ Clean Code
* ✅ SOLID
* ✅ Design Patterns

## Problem
A document management system needs to create new documents based on complex pre-configured templates (contracts, proposals, reports).
The current code recreates objects from scratch, wasting a lot of time on initializations.

## About CarnaCode 2026
The **CarnaCode 2026** challenge consists of implementing all 23 design patterns in real-world scenarios. Throughout the 23 challenges of this journey, participants are exposed to learning and practicing the identification of non-scalable code and solving problems using industry-standard patterns.
